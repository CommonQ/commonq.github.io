## 链表的倒数第k个节点

代码：
```C
Node *find_buttom_kth(Node * ptr,int k){
	if(ptr==NULL||k<=0)
		return NULL;

	Node * fast=ptr;
	Node * slow=ptr;

	while(fast!=NULL&&k--){

		fast=fast->next;
	}

	if(fast==NULL){
		return slow;
	}

	while(fast){
		slow=slow->next;
		fast=fast->next;
	}

	return slow;

}

```
