## 暴力枚举之分数拆分
一个很简单的分数拆分题：  

已知正整数k,找到所有的正整数,x>=y,使得1/k=1/x+1/y
这道题要先求得x或y的值区间：  

```
∵ x >= y  
∴ 1/x <= 1/y  
又∵ 1/k = 1/x + 1/y
∴ 1/k <= 2/y
∴ y<=2k
∵ 1/k = 1/x + 1/y
∴ x=k*y/(y-k)
又∵ x>=1,y>=1
∴ y>k
∴ k<y<=2k
```

分析完之后上代码：

```C
void fraction_split(){

	int k;
	scanf("%d",&k);

	int y;
	for(y=k+1;y<=2*k;y++){
		int x;
```
